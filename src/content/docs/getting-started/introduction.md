## Technology Stack

- **shadcn/ui v2** - Component distribution platform
- **Base UI** - Accessible component primitives (successor to Radix UI)
- **Tailwind CSS v4** - Utility-first CSS framework with CSS-first configuration
- **React 19** - Latest React features
- **TypeScript** - Type-safe components
- **Lucide React** - Beautiful, consistent icons

## Philosophy

We follow the shadcn/ui "open code" philosophy:

- **You own the code** - Components are copied to your project, not installed as dependencies
- **Full control** - Modify components to fit your needs
- **No lock-in** - Use what you need, customize everything
- **Best practices** - Built with modern web standards
